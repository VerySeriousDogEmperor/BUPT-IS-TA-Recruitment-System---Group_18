## 🎯 下一步工作

### 优先级 1：MO 端后端
需要创建以下 Servlet：

1. **MOJobServlet** - 职位管理
   - GET /api/mo/jobs - 获取 MO 的职位列表
   - POST /api/mo/jobs - 创建新职位
   - PUT /api/mo/jobs/{id} - 更新职位
   - DELETE /api/mo/jobs/{id} - 删除职位
   - POST /api/mo/jobs/{id}/submit - 提交职位（发布）

2. **MOApplicantServlet** - 申请人管理
   - GET /api/mo/applicants - 获取申请人列表
   - GET /api/mo/applicants/{id} - 获取申请详情
   - PUT /api/mo/applicants/{id}/status - 更新申请状态（录用/拒绝）

3. **MOTimesheetServlet** - 工时审核
   - GET /api/mo/timesheets - 获取工时表列表
   - PUT /api/mo/timesheets/{id}/review - 审核工时表

4. **MOModuleServlet** - 课程模块
   - GET /api/mo/modules - 获取 MO 的课程列表
   - GET /api/mo/modules/{id} - 获取课程详情

### 优先级 2：Admin 端后端
需要创建以下 Servlet：

1. **AdminJobServlet** - 职位审核
2. **AdminUserServlet** - 用户管理
3. **AdminDashboardServlet** - 数据统计
